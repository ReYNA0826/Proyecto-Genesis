## Símbolo

**La "G" dorada dentro de un círculo, con una estrella** (opción 7 — "neutral,
atemporal y representativo"). Diseñado para trascender generaciones.

| Elemento | Significado |
|----------|-------------|
| **La G** | Origen, crecimiento, guardián, guía y generaciones |
| **El círculo** | Continuidad, eternidad, conocimiento que nunca termina |
| **La estrella** | Nuestro faro. El futuro es brillante |
| **El espacio vacío** | Lo que aún no conocemos. Siempre hay espacio para aprender |
